- Init map w params: width height map_type
  - Invalid params.
  - Double reinit.
  - Deinit-reinit.
  - Double deinit.
  - Read back.
  - Exhaust VRAM.

- Init tileset w params: width height bpp num_tiles
  - Attempt invalid params
  - Attempt to reinit
  - Deinit-reinit
  - Double deinit
  - Read back
  - Exhaust VRAM.

- Init bitmap w params: width height bpp
  - Attempt invalid params
  - Attempt to reinit
  - Deinit-reinit
  - Double deinit
  - Read back
  - Exhaust VRAM.

- Irqline/scanline: Wait until line irq then read scanline
- Turn display off then back on, read back display state
- Modify hscale, read back
- Modify vscale, read back
- Modify screen boundaries, read back

Layers:
- Turn layer off then back on, read back state.
- Modify hscroll
- Modify vscoll
- Install a map and tileset. Params: map_width map_height map_type tile_width tile_height bpp
  - Set tile in four corners
  - Attempt invalid row/col
  - Read back.

- Draw tile. Params width height bpp.
  - Attempt invalid x/y
  - Read back.

- Draw bitmap. Params width heigh bpp.
  - Attempt invalid x/y
  - Read back.

- Modify bitmap palette offset. Read back.

- Modify palette entries. First, last, read back.
  - Restore default palette.
  - Check system palette colors.
  - Use border color. Read back.

Sprite:
- Switch sprite banks, get sprite bank
- Init sprite. Params tileset_id tile_idx
  - Attempt invalid params
  - Read back

- Modify sprite x. Read back.
- Modify sprite y. Read back.
- Modify sprite palette offset. Read back.
- Modify z-depth. Read back.
- Modify h-flip. Read back.
- Modify v-flip. Read back.

- Test collision mask. Read back.

- Draw sprite.
  - Attempt invalid x y
