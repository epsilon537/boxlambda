# The BoxLambda Base SoC Build

- **BoxLambda Base Project in the BoxLambda Directory Tree**:
  [gw/projects/boxlambda-base](https://github.com/epsilon537/boxlambda/tree/develop/gw/projects/boxlambda_base)

This project builds the 'official' BoxLambda Base Configuration, as described in the [SoC Architecture section](../top-level/soc-architecture.md#base-configuration).

This is an FPGA-only project build. This build expects to find a bootloader image to boot from in flash memory.

For instructions to build and flash the BoxLambda Base SoC, see [this section](../../installation/installation.md#installing-the-boxlambda-base-bitstream-bootloader-and-os).
